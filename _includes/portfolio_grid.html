Solar Energy Forecast

Developed a machine learning pipeline to forecast solar energy generation (kWh) from historical radiation and weather data. Implemented feature engineering for seasonal and climatic variables, data normalization, and lag variables. Trained a Random Forest Regressor to achieve reliable energy predictions. Delivered as a complete solution with a FastAPI REST endpoint (/predict_solar) and an interactive Streamlit dashboard for visualization of predictions, metrics, and charts. Entire project was containerized with Docker for reproducibility and deployment in any environment.

Movie Recommendation System

End-to-end content-based recommender built in Python using NLP to suggest similar movies from the TMDB 5000 Movies + Credits dataset. I designed a full data workflow: cleaning (duplicates, missing values), normalization, and merge of Movies/Credits; engineered a soup feature combining keywords, genres, cast, and crew. Developed a content-based recommender using NLP and similarity measures on the TMDB 5000 Movies + Credits dataset. Implemented preprocessing, feature engineering (soup with keywords, genres, cast, crew), and tested different approaches (TF-IDF, CountVectorizer with cosine similarity). Built an interactive Streamlit app where users can search movies and get personalized recommendations. The system was Dockerized for easy deployment, with a roadmap for future improvements like integrating posters via TMDB API and hybrid recommendation models.

Flower Classifier

Built a deep learning image classifier for five flower species using TensorFlow/Keras. Implemented a full data pipeline with image loading, normalization, caching, shuffling, and prefetching for efficient GPU training. Leveraged transfer learning with EfficientNetB0 pretrained on ImageNet, achieving 92% validation accuracy. Implemented model saving and visual prediction outputs with probabilities, making the project suitable for both educational and practical use in computer vision tasks.

Library Management System

Personal project written in Python to explore data structures, algorithms, and object-oriented programming. Includes features to add, search, borrow, and return books, with automatic status updates (“available” or “borrowed”). Implements a dynamic menu-driven interface to handle user interaction, error handling, and iterative program flow until exit. Designed as a study case to simulate a real-world library system, reinforcing knowledge of collections, loops, and conditionals.
